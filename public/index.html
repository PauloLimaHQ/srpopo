Autonomous Mode

Sr. Popo will drive every Ready task in this workspace end-to-end — dispatch each run (which plans, builds, tests, self-reviews, and opens a PR), then merge the PR and move the task to Done once it’s green — until the budget is spent or the queue is empty. Runs are unattended: tool-permission prompts are turned off for them, so make sure each task is configured to run without a human.

Brief an Idea

Describe a rough idea and pick a project. Sr. Popo runs a quick, read-only Claude session in that repo to groom it into a well-formed task prompt, then drops it in Ready.

Pick a project, then paste a Linear issue ID or choose one of your assigned issues. Sr. Popo pulls the issue and grooms it into a well-formed task in Ready.
